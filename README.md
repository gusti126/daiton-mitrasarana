## Build Setup

```bash
# download
$ git clone https://github.com/gusti126/daiton-mitrasarana.git
$ daiton-mitrasarana
$ cp .env.example .env

# install dependencies
$ composer install
$ npm install

# set your database in .env & run
$ php artisan migrate

# if your need data faker
$ php artisan migrate:fresh --seed

# build launch server
$ php artisan serve
$ npm run watch
```
