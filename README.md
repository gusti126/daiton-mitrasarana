## Build Setup

### download

```bash
git clone https://github.com/gusti126/daiton-mitrasarana.git
```

```bash
cd daiton-mitrasarana
```

```bash
cp .env.example .env
```

```bash
cp .env.example .env
```

### instal dependency

```bash
composer install
```

```bash
npm install
```

### set your database in .env & run

```bash
php artisan migrate
```

### if you need data faker

```bash
php artisan migrate:fresh --seed
```

### build launch server

```bash
php artisan serve
```

```bash
npm run watch
```
